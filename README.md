# Tiomila Results Search

A simple web application to search and explore historical Tiomila orienteering results.

## How to Run Locally

Since this is a static website, you need a local web server to view it correctly (to handle file fetching and other browser security features).

Run:

```powershell
python -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

## Features

- Search for clubs and view their historical performance.
- Direct links to original event results (Eventor/Winsplits).
- Mobile-responsive design with dark mode.
- Donation button for project support.
